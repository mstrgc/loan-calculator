# Rade Loan Calculator Plugin

a wordpress plugin that uses php and ajax to help you calculate bank loan payments and required deposit, by providing a shortcode that could be used anywhere in website.

## How to use

1. Upload the plugin folder to /wp-content/plugins/
2. Activate plugin from wordpress plugins admin menu
3. Place the shortcode whereever you need the calculator

## Example:
```
[loan_calculator bank_name="melli"]
```
available attributes so far: 'melli', 'mehr'.
